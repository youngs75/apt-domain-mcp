## 4. 개정 이력 스토리 (Phase 1에서 v2/v3 합성)

본 파일럿의 개정 이력은 "현실적인 공동주택 개정 패턴"을 담는다:

### v1 → v2 (가상 2020-06-01 개정)
- **제41조(관리비 부과기준)**: 세대별 산정 기준표 변경 (공용전기 배분방식 세대수 → 전용면적 비례)
- **제68조(반려동물)**: 신설. 이전 규약엔 없던 조항을 추가.
- **제55조(주차장)**: 방문차량 무료 시간 2시간 → 1시간 단축

### v2 → v3 (가상 2023-04-01 개정)
- **제15조(대표회의 정원)**: 25명 → 21명 (세대수 감소 반영)
- **제48조(장기수선충당금)**: 요율 조정
- **제70조(층간소음)**: 신설. 2023년 정부 가이드라인 반영.

## 7. 라이선스·출처 표기

가상 단지 산출물은 모두 **본 리포에서 합성된 허구**이며, 실존 법령·준칙을 구조적으로만 참조한다. 조문 문구는 공동주택관리법·서울시 준칙의 공개 문안을 참고한 재작성이며, 실제 단지에서의 법적 효력을 갖지 않는다.

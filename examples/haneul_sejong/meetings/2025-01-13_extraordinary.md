# 세종 하늘마을 아파트 입주자대표회의 회의록

- **회의번호**: 제2025-ex01차 임시회의
- **일시**: 2025년 1월 13일 (월) 20:00 ~ 22:40 (온·오프라인 병행)
- **장소**: 커뮤니티센터 4층 다목적홀 + 공유 플랫폼 화상회의
- **유형**: 임시 (개인정보 침해 의심 사고 대응)
- **정원**: 41명 (v2 기준)
- **참석**: 38명 (오프라인 26명, 온라인 12명) / 재적 41명, 과반수 21명 충족
- **불참**: 3명 (긴급 업무 1명, 해외 체류 2명)
- **배석**: 관리사무소장, 스마트운영팀장, 개인정보 처리담당자, 외부 보안 컨설턴트 정○○ (의뢰 조사 결과 설명)
- **의결정족수**: 출석 과반수 20명
- **서기**: 관리사무소 총무 이선아

> 본 회의록은 `apt-domain-mcp` 예시용 합성 허구이며 실제 법적 효력이 없다.

## 안건 1. 공유 플랫폼 개인정보 침해 의심 사고 조사 결과 및 대응의 건

### 제안 배경
2025년 1월 3일 05:12 공유 플랫폼 서버 모니터링에서 비정상 접근 시도(외부 IP, 예약·민원 API) 가 탐지되었고, 1월 4일 긴급 차단 조치 후 외부 보안 컨설턴트 정○○의 침해 여부 조사를 의뢰하였다. 본 회의는 조사 결과 공유와 후속 대응 결정을 위한 임시 소집.

### 주요 토의
- 보안 컨설턴트 정○○: 1월 2~3일 시도된 접근은 외부 자동화 봇에 의한 엔드포인트 스캔 수준. 실제 개인정보 탈취는 확인되지 않았으나, 예약 API 중 일부가 인증 토큰 만료 검증이 느슨해 **12건의 예약 내역이 외부에서 조회 가능했던 시간 창이 약 6시간 존재**. 실제 외부 조회는 0건(로그 검증).
- 개인정보 처리담당자: 제86조제6항(v3 후보) 에 준하는 대응 프로세스를 선제 적용. 1월 4일 관련 입주자등 12세대에 개별 통지 완료. 관계 기관(개인정보 보호위원회) 에는 1월 8일 서면 신고 완료(법적 신고 대상은 아니나 투명성 확보 목적).
- 감사: 사고 대응은 적절했으나, **사고 인지 시점과 이용자 통지 시점 사이에 24시간 이상 간격**이 있었던 점이 향후 개선 과제. 침해 의심 단계에서 영향 세대에 사전 고지 원칙을 운영규정에 추가할 필요.
- 이사 N: 제86조(데이터 프라이버시) 신설을 포함한 v3 개정을 2월 정기회의에서 원안대로 제안하는 것이 본 사고의 결론을 규약에 반영하는 길. 회장 동의.
- 스마트운영팀장: 예약 API 인증 토큰 검증 강화 패치 배포 완료(1월 6일). 외부 침투 테스트 연 1회에서 **연 4회(분기별)** 로 증액 제안.

### 결정사항
- **가결(1)**. 본 사고에 대한 외부 보안 컨설턴트 조사 결과를 수락한다. 실제 개인정보 탈취 없음 확인.
- **가결(2)**. v3 개정 안건(제86조 데이터 프라이버시 신설 + 제61조·제68조 개정)을 2월 18일 정기회의에 원안 상정한다.
- **가결(3)**. 외부 침투 테스트를 분기별 실시로 확대하며 필요 예산을 2025년 예산에 반영한다(연 약 1,800만원 증액).
- **가결(4)**. 운영규정에 **침해 의심 단계에서의 사전 고지 원칙**을 추가하고, 2월 말까지 개정 완료.
- **투표**: 찬성 37 / 반대 0 / 기권 1 (의결 4건 일괄)
- **관련 조문**: 제6조, 제56조제4항(v2), 제85조(v2), 제86조(v3 후보), 제33조제6호
- **후속조치**: 개인정보 처리담당자는 1월 20일까지 영향 세대 재통지 완료, 사고 종결 보고서 공유 플랫폼에 공개. 스마트운영팀장은 1월 말까지 운영규정 개정안 작성.

## 안건 2. 2025년 예산 수정 (보안 강화 반영)의 건

### 제안 배경
안건 1 의결 결과 외부 침투 테스트 예산 1,800만원 증액이 필요. 2024년 11월 정기회의에서 승인된 2025년 예산안을 일부 수정해야 한다.

### 주요 토의
- 회계담당자: 2025년 총 예산 규모 47억 6,200만원. 보안 증액 1,800만원은 전체의 0.04% 수준. 예비비(3%)를 활용하거나, 공유 플랫폼 라이선스 연장 협상분(기대 절감 2,100만원)을 재원으로 활용 가능.
- 감사: 플랫폼 라이선스 절감분은 미확정이므로, 1차적으로 예비비로 충당하고 절감분 확정 시 예비비로 환입하는 이중 안전판 권장.
- 이사 P: 본 보안 증액은 "침해 사고 대응"이라는 명시적 사유가 있으므로, 입주자등에 대한 공유 플랫폼 공지에 예산 재분배 내역을 투명하게 포함할 것.

### 결정사항
- **가결**. 외부 침투 테스트 예산 1,800만원 증액을 2025년 예산에 반영하며, 1차 재원은 예비비로 한다. 공유 플랫폼 라이선스 절감분 확정 시 예비비로 환입.
- **투표**: 찬성 38 / 반대 0 / 기권 0
- **관련 조문**: 제21조제1호, 제45조, 제47조
- **후속조치**: 회계담당자는 1월 20일까지 수정 예산서 게시.
